# Mother Mary Public School Website

A modern, responsive, and SEO-friendly frontend website developed for Mother Mary Public School. The project focuses on delivering a clean user interface, smooth animations, fast performance, and an excellent experience across desktop, tablet, and mobile devices.

## Features
- Responsive Design
- SEO-Friendly Structure
- Modern UI/UX
- Smooth Animations
- Home, About, Academics, Gallery, Admissions & Contact Pages
- Cross-Browser Compatible
- Fast Loading
